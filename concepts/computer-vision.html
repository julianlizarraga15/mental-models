Parent concept

Computer Vision

Computer vision is the field of making machines extract useful structure from images and video.

Input
Image, video frame, scanned document, blueprint.
↓
Computer vision task
Detection, segmentation, OCR, classification, layout analysis.
↓
Output
Labels, boxes, masks, text, tables, key-value pairs.

Simple mental model

Computer vision is not one task. It is a toolbox. Different tasks produce different outputs. A detector gives boxes. A segmenter gives masks. OCR gives text. A layout model gives page structure.

Inside computer vision

Object Detection

Find objects and draw boxes around them.

Segmentation

Assign pixels to objects, regions, or classes.

OCR

Turn visual text into machine-readable text.
