DETR

DETR treats detection as a set of slots. Each slot predicts one object — or nothing.

1) Queries = object slots

2) Each slot predicts one object — or no object

3) Training matches predictions to real objects

Why no anchors?

The slots directly predict boxes. They do not start from preset anchor boxes.

Why no NMS?

Because training already pushes the model toward one prediction per object.

Mental model

DETR is object detection as set prediction, not box cleanup.
