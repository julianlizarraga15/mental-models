Object Detection

Object detection answers: what objects are present, and where are they?

Image
A blueprint, street photo, CCTV frame, document page.
↓
Detector
YOLO, DETR, RT-DETR, RF-DETR, etc.
↓
Boxes + labels
water closet: 92%, sink: 88%, door: 76%

Simple mental model

Detection is about rectangles. The model does not fully understand the object shape. It says: “I think this thing is inside this box.”

When it is useful

Use detection when a bounding box is enough: counting objects, locating symbols, finding fixtures, detecting people/cars, or cropping regions for later processing.
