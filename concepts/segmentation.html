Concept

Segmentation

Segmentation answers: which pixels belong to which thing?

Semantic segmentation
Each pixel gets a class: road, wall, person, table.
Instance segmentation
Each object instance gets its own mask: person 1, person 2.
Panoptic segmentation
Combines class-level regions and individual object instances.

Simple mental model

Detection draws boxes. Segmentation paints the actual shape. If object boundaries matter, segmentation is usually better.

When it is useful

Use segmentation when the exact region matters: document layout regions, object shape, medical imaging, floorplans, roads, masks, or separating overlapping objects.
